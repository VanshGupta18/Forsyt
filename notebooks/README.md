# Notebooks

Exploratory analysis and one-off research notebooks.
Not part of the production pipeline — keep experimental code here.

## Suggested notebooks

| Notebook | Purpose |
|---|---|
| `01_eda_gpr_history.ipynb` | EDA on historical GPR vs Caldara benchmark |
| `02_feature_analysis.ipynb` | Feature correlation heatmaps, distribution plots |
| `03_model_comparison.ipynb` | LR vs XGBoost vs LSTM comparison plots |
| `04_portfolio_analysis.ipynb` | Sector beta visualisations |
